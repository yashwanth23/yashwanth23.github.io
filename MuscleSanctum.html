Muscle Sanctum

Creative Technology | User Experience Design | Gamification

ROLE

Developer, Experience Designer, Animator

TEAM

Sole Contributor

SKILLS

Unity game programming, Kinect programming, Level design, Motion graphics, Instructional design, User experience design, Motion capture

TOOLS

Unity, Kinect, After effects, Photoshop

KEYWORDS

Immersive fitness, Muscle workouts, Exergame, Gamification, Kinect game

DESCRIPTION

Muscle Sanctum is an interactive visual interface system for novice fitness enthusiasts to learn about muscle workout exercises in a fun and effective manner. The project aims to gamify a routine experience such as working out and make it engaging for the users, leading to longer involvement with the experience. By making use of instructional design, game design and user experience design principles, the project creates a holistic environment for such workouts to take place. Motion capture and depth-sensing technologies are used to recognize and compare body movements in real-time and provide necessary feedback to the user while performing. Unlike fat burning exercises, these workouts have to be performed with focus and at a much slower pace; hence this system debunks the common notion of fitness being an energy induced fast pace workout. The ultimate goal of this project is to give the users necessary training in order for them to advance to the weights and machines section in the gym.



RESEARCH

Strength training/Muscle workout exercises unlike fat burning exercises often requires the use of weights to challenge your muscles in order to increase muscle mass, toned muscles and strengthened bones. These workouts are often performed using free weights, resistance machines and body weights. It is recommended to perform these exercises at a slower and controlled rate rather than explosive movements in order to reduce the risk of injury and maintain muslce tension. In order to achieve maximum benefits of these exercises it is suggested to perform them with certain guidelines. The figure below shows some important tips for safe and effective workout.

Since the project emphasizes on teaching exercises in an effective way, instructional design principals are used. One of the strategies that recently impacted a lot on the learning environment is the use of games or simulations. There have been many studies to show that by including some type of game within a course that the retention is greater by the learner. Another benefit of using this type of activity within training is that it is fun. When learners experience fun in training the desire to learn more increases and the process of learning is greatly improved. Similarly in the scenario of teaching workouts, games could be an effective tool to encourage learning and retention. The purpose of this project was to verify if interactive games can engage the audience and make them involved in learning exercises for a longer term.




USER PERSONA AND SCENARIO




The larger scope of the system is two-fold. The primary aspect is a screen interface experience for users to learn exercises in real time through constant feedback and the secondary aspect is a mobile application to track progress and notify users



PROTOTYPES

Visual interpretations of the above mentioned guidelines were made as prototypes to test if users were able to follow them intuitively without prompts. These prototypes are made using arduino, tilt sensors and are coded in p5.js, posenet and Unity.

The user tests of the prototypes turned out positive as users were able to follow along while also having fun. The next step in the process was to bring all the different prototypes together as various visual layers in a single experience. Unity game engine was used to recreate the visual interpretations from the previous prototypes and create a single coherent scene.



FINAL PRODUCT

Furthermore, in order to create a dynamic and immersive experience, Unity is used to create whole virtual ecosystem, establish connection with depth sensor and enact user experiences within the product.



Virtual world design: To create an immersive environment, level design is done in Unity. Forest environment and meditative theme is used. The idea behind this is to bring a sense of calmness and focus to the user while performing these exercises. Muscle workouts unlike fat burning workouts needs to be done at a slower controlled pace and with concentration, hence the forest and meditative theme.

Technology choices: In terms of technology there are two aspects, one is the virtual trainer which replicates the movements of a certified trainer captured in Motion capture studio and the other is the user, the motion of which is captured using Kinectâ€™s skeleton tracking in real-time. Pose estimation and gesture recognition algorithms are used to tally these data points and provide real-time feedback.

User experience (Instructional design): Since the core of this project is about teaching exercises, instructionl design principles are used to create a learning experience. Using Kemp's model of instructional design, a step by step instruction model has been developed which is integrated with the experience.

User experience (Interactions): The interactions in this project are mainly through body movements. Game design principles are used to create visual cues and feedback mechanisms that are intuitive for users to follow along. The most common ones are integrated, emphasized and overlaid experiences that allow the users to understand the gameplay while also perfectly perform exercises.
